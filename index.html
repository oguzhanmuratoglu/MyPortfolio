E-Trade Website

.NET

This site is built with 100% .NET architecture. Layered architecture is used in backend codes. The .net libraries Identity, UserManager, RoleManager and SignInManager were used in page coding. MVC and Bootstrap were used for the frontend part. In the user panel on the site: can add products to the basket, switch between categories and perform basket transactions. In the admin panel, there is an area where you can add, remove and update products. Each admin logs in to its own specific area and without admin login, admin page loading will fail.
